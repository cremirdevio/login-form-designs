## Basic Form Design

Basic form design using HTML and CSS

## Input Attributes
1. `name` - The unique identifier of the input
1. `autocomplete`
1. `value`
1. `placeholder` - Description of the input
1. `autofocus`
1. `disabled` - Form can't be edited
